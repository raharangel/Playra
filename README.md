📌 Project Overview  
Playra is a web application designed to explore and display game-related data in an interactive way.  
The application fetches real-time data from a public API and allows users to search and view information dynamically.

🔰 Introduction  
The objective of this project is to build a web application that demonstrates understanding of JavaScript, API integration, and UI development.  

The application includes:  
- Integration of a public API using fetch()  
- Use of array higher-order functions (filter) for searching  
- A clean and responsive user interface using CSS  


🔗 API Used  
https://pokeapi.co/api/v2/pokemon  

This API provides data about Pokémon such as names and details.  
It was selected because it is simple, does not require authentication, and is useful for learning API integration concepts.
